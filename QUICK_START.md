# Быстрый старт

## Проблема с npm в PowerShell?

Если npm не распознаётся в PowerShell, используйте один из способов ниже.

## Решение 1: Использовать скрипт (Самый простой способ)

Просто запустите:
```powershell
.\start.ps1
```

Этот скрипт автоматически обновит PATH и запустит сервер.

## Решение 2: Обновить PATH вручную

В PowerShell выполните:
```powershell
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
npm run dev
```

## Решение 3: Использовать обычную командную строку (cmd)

Откройте обычную командную строку (cmd) вместо PowerShell:
```cmd
cd D:\exampleFront
npm install
npm run dev
```

## После запуска

1. Откройте браузер
2. Перейдите на http://localhost:3000
3. Вы должны увидеть сайт с счётчиком

## Если сайт не открывается

1. Проверьте, что сервер запущен (в терминале должно быть сообщение "Local: http://localhost:3000")
2. Попробуйте открыть http://127.0.0.1:3000
3. Проверьте консоль браузера (F12) на наличие ошибок
4. Убедитесь, что порт 3000 не занят другим приложением
